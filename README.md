# Fundamental Programs Repository  

Welcome to the **Fundamental Programs** repository! 🚀 This collection of Java programs is designed to help beginners and aspiring developers strengthen their understanding of core programming concepts through practical coding exercises.  

## 🔎 Project Overview  
The **Fundamental Programs** repository covers essential Java logic and algorithm development. It includes a variety of fundamental programs categorized into:  

- **Mathematical Operations** (Palindrome, Fibonacci, Prime numbers, Factorial)  
- **Array Manipulation** (Reverse array, Find largest and smallest elements)  
- **String Operations** (Reversal, Anagram check, Vowel/consonant count)

## 🧩 What You'll Learn  
- Implementing fundamental algorithms using Java.  
- Strengthening problem-solving skills with logical exercises.  
- Enhancing understanding of loops, conditionals, recursion, and data structures.  

## 📄 Programs Available  

### 🏆 Mathematical Programs  
- Palindrome check  
- Fibonacci series  
- Prime number detection  
- Factorial calculation  

### 🔢 Array Programs  
- Reverse an array  
- Find the largest and smallest element  

### 🔡 String Programs  
- Reverse a string  
- Check if two strings are anagrams  
- Count vowels and consonants  
- Convert uppercase to lowercase and vice versa    

## 🎯 Who Can Benefit?  
✅ **Beginners** – Perfect for those new to Java and looking to build a strong foundation.  
✅ **Programming Students** – Excellent for practicing common coding problems and improving logical thinking.  
✅ **Self-Learners** – Ideal for reinforcing coding skills with hands-on exercises.
